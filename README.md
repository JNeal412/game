# game
# 接球大挑战 - 终极豪华版 v3.0
一款基于 Pygame 开发的多模式休闲游戏，支持无尽、限时、挑战三种玩法，包含道具系统、皮肤商店、成就系统等丰富功能。

## 功能特点
- 三种游戏模式：无尽模式、限时模式、挑战模式
- 10+ 种道具效果：加分、减速、护盾、磁铁、双倍分数等
- 7 种挡板/球皮肤，可通过分数解锁
- 成就系统与排行榜记录最高分
- 流畅的粒子特效与动态背景

## 运行方式
1.  安装依赖：`pip install pygame pillow`
2.  运行游戏：`python game.`

## 操作说明
- `←` `→` 或 `A` `D`：移动挡板
- `ESC`：暂停/继续游戏
